listar los plugins instalados
phonegap plugins list

phonegap plugin remove NOMBREDELPLUGIN

si no deja eliminar el plugin 
phonegap platform list
phonegap platform remove browser

para instalar el plugin
phonegap plugin add NOMBREDELPLUGIN
